Celestic_clothesitems is ispired by keep-backpack.
The script has been completely rewirtten into clothing-items with bag slots.
The idea came to me in the evening an then i sat down right away. 
Because who doesn't want to have clothes as items where you can store items????


In forqbcoreitems you will find the code that goes into the : shared/items.lua , 
weight/images are subject to change as well as names.

The description from the items can also be adapted in each case
in forqbshops you will find the shops.lua there are shops for male and female,
so that everything is divided and not in one shop!.

in Inventoryitemphotos you will find inventory pictures for the items.
division is 
- female/done - in done you will find the finished pictures and in backup the original pictures.
- male/done - in done you will find the finished pictures and in backup the original pictures.
